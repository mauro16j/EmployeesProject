Bienvenido a la Aplicacion Angular que consume los servicios REST de Employee para Mas Global

Por Favor dirigirse al menu Fetch Data para iniciar la aplicacion.

Contiene las siguientes caracteristicas:

  • Proyecto de Test
  • Front End en Angular
  • Api Rest en .Net Core 2.1 VS 2017
  • BLL Layer
  • DAL Layer
  • Model

Contiene los siguientes patrones de diseño:

  • DI
  • Factory

The ClientApp subdirectory is a standard Angular CLI application. If you open a command prompt in that directory, you can run any ng command (e.g., ng test), or use npm to install extra packages into it.
